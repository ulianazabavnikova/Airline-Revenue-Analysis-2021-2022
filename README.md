# Airline-Revenue-Analysis-2021-2022
Исследование факторов, влияющих на выручку авиакомпании: анализ сезонности, поведения клиентов (ранние/поздние бронирования) и популярных направлений. Проведена статистическая проверка гипотез, корреляционный анализ, изучены различия в выручке по типу пассажиров, каналу продаж и программе лояльности.
